# 🧠 FEATURE OVERVIEW

Users can create **anonymous posts** where:

* Their identity is hidden
* Only:

  * Program (e.g., BCA)
  * Year (e.g., 2nd Year)
    is visible

---

# 📝 PART 1: CREATE ANONYMOUS POST

---

## POST CREATION:

Add toggle:
👉 "Post Anonymously"

---

## IF ENABLED:

Store in DB:

* post content
* media (if any)
* program
* year
* user ID (hidden internally)

---

## DISPLAY:

Instead of name:

👉 Show:

* "Profile Hidden"
* Program + Year

Example:
"Profile Hidden • BCA 2nd Year"

---

## OTHER USERS SEE:

* Post content
* Comments
* Replies
* Program + Year

---

## OTHER USERS DO NOT SEE:

❌ Username
❌ Profile image
❌ Any identity
